# fb-like-ui
Simple implementation of facebook like <i>compose post and display</i> UI. You can add gifs in post using giphy api.

## Development
1. Clone this repo.
```shell
  git clone https://github.com/saurabh-33/fb-like-ui 
```

2. Install dependencies.
```shell
  cd fb-like-ui
  npm i
``` 

3. Start development server.
```shell
  npm run dev
```

4. Build for production.
```shell
  npm run build
```
